## Disclosure Policy

Please DO NOT:

- Publicly disclose vulnerabilities before they are reviewed
- Attempt to exploit vulnerabilities beyond proof-of-concept
- Perform denial-of-service testing

Responsible disclosure will be acknowledged.

---

## Security Best Practices Used

This repository follows these practices:

- No API keys committed to version control
- Use of environment variables for credentials
- Educational-only financial examples
- Regular dependency updates
- Minimal third-party scripts in blog pages
