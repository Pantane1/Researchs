Objective

Design and implement an AI-based trading system targeting BTC/USDT. The system predicts short-term price direction using machine learning and executes trades via Binance API with risk management.

Risk Management

  • 1–2% capital per trade
  • ATR-based stop loss
  • 2:1 risk-reward ratio
  • Max 3 concurrent trades
